Installation Filetype Hider

Open Command Prompt as Administrator

Type: " cd C:\Windows\Microsoft.NET\Framework\v4.0.30319 "

Next search the "ExplorerFiletypeService.exe" file inside the folder.

It should be located under: ExplorerFiletypeService\bin\Debug\ExplorerFiletypeService.exe

Right click and copy the path. Head back into Command Prompt.

Type: " InstallUtil.exe [CTRL+V / save path] "

Now it installs the service.


To setup the tool, head to:

C:\Users\Public\Documents\FourteenDynamics\ExplorerFiletypeHandler

Now, open up the "ExtentionsToHide.txt"

In here, you can define which types should be hidden. Create a new line and write "*.Filetype"

Create a new file of the type you just defined, and wait a few moments (the interval is set to 5 seconds).

Now you should see that the file has disappeard. You can open up the "HiddenFiles.txt" file to see all files that have been hidden. DO NOT DELETE THIS FILE

If you want to see the Files anyways, you can just show hidden files through the explorer.


In case the service doesn't run, click the windows button and search "Services". Open it up and scroll down to "Explorer Filetype Handler".

Start the service. Here, you can also define if the service should automatically start when windows boots.


To show all hidden files, stop the service, start it again, and directly stop it again.

Everytime the service starts, it will show all proviously hidden elements.


Uninstallation:

Open Command Prompt as Administrator

Type: " cd C:\Windows\Microsoft.NET\Framework\v4.0.30319 "

Next search the "ExplorerFiletypeService.exe" file inside the folder.

It should be located under: ExplorerFiletypeService\bin\Debug\ExplorerFiletypeService.exe

Right click and copy the path. Head back into Command Prompt.

Type: " InstallUtil.exe -u [CTRL+V / save path] "
